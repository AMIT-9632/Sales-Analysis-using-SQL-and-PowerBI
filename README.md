# AdventureWorks Data Analysis with SQL and Power BI

This repository contains an end-to-end data analytics project built on the AdventureWorks dataset. The project focuses on cleaning and analyzing the dataset with SQL, then transforming the results into interactive dashboards and visual reports in Power BI .

## Project Overview

The AdventureWorks database is a Microsoft sample dataset commonly used for SQL Server learning, analytics practice, and business intelligence projects . In this project, the dataset is prepared through SQL-based cleaning and analysis, and the final insights are presented through Power BI dashboards for easier business interpretation.

## Objectives

- Clean and standardize raw AdventureWorks data using SQL.
- Perform exploratory and business-focused analysis on key entities such as sales, products, customers, and performance trends .
- Build visual dashboards in Power BI to communicate insights clearly and interactively.
- Create a project that demonstrates practical skills in data cleaning, querying, reporting, and storytelling with data .

## Tools Used

- **SQL / SQL Server** for data cleaning, transformation, and analysis .
- **Power BI** for data modeling, dashboard building, and visualization.
- **AdventureWorks** sample database as the project dataset .

## Workflow

### 1. Data Cleaning in SQL

The first stage of the project focuses on preparing the AdventureWorks dataset for analysis. This typically includes inspecting tables, handling inconsistencies, selecting relevant columns, and writing SQL queries to produce structured analytical outputs [web:2][web:4].

### 2. Data Analysis in SQL

After cleaning, SQL is used to analyze patterns in the data. Typical analysis areas include sales performance, customer behavior, product trends, and other business metrics that can support decision-making [web:2].

### 3. Visualization in Power BI

The cleaned and analyzed data is then imported into Power BI to build dashboards. Power BI enables interactive filtering, visual comparison, and a more intuitive understanding of trends and performance across different business dimensions [web:2].

## How to Run

1. Download and install the AdventureWorks sample database for SQL Server 
2. Open the SQL scripts in the `sql/` folder and run the cleaning and analysis queries against the database.
3. Export or connect the query outputs to Power BI.
4. Open the Power BI file and refresh the data model.
5. Explore the dashboards and insights.
